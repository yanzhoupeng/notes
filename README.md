# Notes

基于 Obsidian+Docsify+Github Pages 的个人笔记，用于整理学习的知识。

## 搭建步骤

1. 本地可以使用 obsidian 或者任意其他笔记软件进行笔记管理，在软件根目录进行 docsify 初始化
2. 使用 Git 对笔记仓库进行初始化，并将该笔记仓库关联到 GitHub 仓库
3. 参考 docsify 官网进行本地配置
4. 开启 GitHub page
